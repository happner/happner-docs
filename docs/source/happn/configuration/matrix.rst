Configuring happn and happner
=============================

The happn and happner frameworks utilise a comprehensive configuration system, which can be overwhelming to understand
and set up. The **happn-configuration** library provides a mechanism whereby configuration files can be generated using
a *builder* class (exported as the :code:`ConfigBuilderFactory`).

**happn-configuration** is a Typescript project, however this has been published to *npm* as a set of CommonJS modules (the
published package contains Typescript definition files to allow for type safety and type-ahead if you are developing in
Typescript).

The :code:`ConfigBuilderFactory` class exposes the following set of functions, to allow a fluent (chaining) approach to
building configuration files.

Please note the following:

- function parameters are omitted for brevity (see links to function details)
- the table key is:
   - H: happn
   - HR: happner
   - HC: happn-cluster
   - HRC: happner-cluster
